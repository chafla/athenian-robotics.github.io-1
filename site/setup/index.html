Machine Setup

OSX

Install the following pieces of software:

  • XCode

  • Java 8

  • brew

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  • Python 3
$ brew install python3
  • git
$ brew install git
  • maven
$ brew install maven
  • gradle
$ brew install gradle
  • httpie
$ brew install httpie

Periodically verify that brew is configured properly with: bash $ brew doctor
